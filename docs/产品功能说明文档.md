# CRM 客户关系管理系统 — 产品功能说明文档

**版本**：v1.0.0  
**更新日期**：2025年

---

## 一、产品概述

CRM 客户关系管理系统是一套面向中小企业的全流程销售管理平台，覆盖从市场线索获取到合同签署、款项回收的完整销售链路，帮助销售团队提升效率，管理层实时掌握业务动态。

### 核心价值

- **全流程管理**：线索 → 客户 → 商机 → 合同 → 回款，环环相扣
- **团队协作**：多用户协同，清晰的权限分工
- **数据洞察**：多维度报表，辅助管理层决策
- **简洁易用**：现代化界面，学习成本低

## 二、功能模块详情

### 2.1 工作台（Dashboard）

系统首页，展示关键业务指标与可视化图表。

**核心指标卡**
| 指标 | 说明 |
|------|------|
| 线索总数 | 系统线索总量及本月新增数 |
| 客户总数 | 正式客户总量及本月新增数 |
| 进行中商机 | 活跃商机数量及金额汇总 |
| 本月回款 | 本月已确认回款金额 |
| 签约合同 | 合同总数及本月新签数 |
| 进行中项目 | 进行中项目数及逾期预警 |

**图表分析**
- 近12个月回款趋势折线图
- 客户行业分布环形图
- 销售漏斗图（各阶段商机数量与金额）
- 员工业绩对比柱状图

---

### 2.2 线索管理

管理潜在客户信息，跟踪线索状态，支持转化为正式客户。

**线索状态流转**
```
新线索 → 已联系 → 已确认 → 已转化
                         ↘ 已失效
```

**核心功能**
- 线索列表查询（支持按名称、公司、联系人搜索）
- 多维度筛选（状态、来源）
- 新建/编辑线索（公司信息、联系方式、行业地区）
- **一键转化**：将线索直接转化为正式客户
- 删除线索

**字段说明**
| 字段 | 说明 |
|------|------|
| 线索名称 | 必填，线索简要描述 |
| 公司名称 | 所属公司 |
| 联系人 | 对接人姓名及联系方式 |
| 来源 | 官网/展会/推荐/电话/广告等 |
| 状态 | 新线索/已联系/已确认/已转化/已失效 |
| 行业/地区 | 分类标签 |

---

### 2.3 客户管理

管理正式客户档案，支持多联系人记录和跟进历史。

**客户级别**：普通 / 重要 / VIP  
**客户状态**：正常 / 非活跃 / 黑名单

**核心功能**
- 客户列表（支持搜索、级别/状态/行业筛选）
- 客户详情页（基本信息 + 联系人列表 + 跟进记录）
- 新建/编辑客户信息
- 添加多联系人（姓名、职位、电话、邮箱）
- **跟进记录**：记录每次沟通内容，支持电话/拜访/邮件/微信/会议等方式，可设置下次跟进时间

---

### 2.4 商机管理

跟踪销售机会，推进成单进程，支持列表和看板两种视图。

**商机阶段（销售漏斗）**
| 阶段 | 默认赢率 | 说明 |
|------|---------|------|
| 意向挖掘 | 10% | 初步发现客户意向 |
| 需求确认 | 30% | 客户需求明确 |
| 方案报价 | 50% | 方案已提交 |
| 商务谈判 | 70% | 进入合同谈判 |
| 赢单 | 100% | 成功签约 |
| 输单 | 0% | 未能成交 |

**核心功能**
- **列表视图**：标准表格，支持排序和筛选
- **看板视图**：按销售阶段分列，直观展示商机分布
- 关联客户和产品/服务信息
- 设置预计成交日期和赢率
- 跟进记录和活动日志

---

### 2.5 项目管理

管理售后项目交付，跟踪项目进度和任务执行。

**项目状态**：规划中 / 进行中 / 已完成 / 已暂停 / 已取消  
**优先级**：低 / 普通 / 高 / 紧急

**核心功能**
- 项目列表（支持状态筛选，自动生成项目编号）
- 项目详情（基本信息 + 任务列表）
- 逾期预警（截止日期已过且未完成的项目高亮显示）
- **任务管理**：
  - 创建任务并分配成员
  - 设置任务状态（待处理/进行中/已完成/已取消）
  - 记录任务完成进度（0-100%）
  - 优先级设置
- 整体进度可视化进度条

---

### 2.6 合同管理

全流程合同管理，包含审批流程和回款计划。

**合同状态流转**
```
草稿 → 审批中 → 已签署（已通过）
              ↘ 已驳回（重新修改后可再次提交）
已签署 → 已到期（超过结束日期）
```

**合同类型**：销售合同 / 服务合同 / 框架合同 / 补充协议

**核心功能**
- 合同列表（编号、名称、金额、回款状态）
- 关联客户和商机
- **审批功能**：审批中状态可一键通过或驳回
- 合同详情：查看回款记录和回款进度
- 自动计算已回款金额和待回款金额

---

### 2.7 回款管理

记录和管理合同回款，支持财务二次确认。

**回款状态**：待确认 / 已确认  
**收款方式**：银行转账 / 支票 / 现金 / 网络支付

**核心功能**
- 回款列表（自动生成回款单号）
- 多条件筛选（状态、收款方式、日期范围）
- 关联合同和客户
- **财务确认**：待确认状态可一键确认到账
- 统计看板：已确认金额、待确认金额实时显示

---

### 2.8 报表统计

多维度数据分析，4个报表页面。

**销售概览**
- 商机阶段分布柱状图
- 销售漏斗可视化
- 线索来源转化率分析表

**客户分析**
- 客户来源分布饼图
- 客户行业分布横向柱状图
- 月度新增客户趋势折线图

**回款分析**
- 近12个月回款趋势面积图
- 月度回款明细表（含环比数据）

**员工业绩**
- 赢单金额 vs 回款金额对比柱状图
- 员工业绩明细排行榜（含排名徽章）

---

### 2.9 系统管理

#### 用户管理
- 用户列表（搜索、状态/角色筛选）
- 新建用户（设置用户名、密码、角色、部门）
- 编辑用户信息和状态（启用/禁用）
- 重置用户密码

#### 角色管理
- 角色卡片展示（含使用人数）
- 新建/编辑角色及权限配置
- **细粒度权限**：8个功能模块 × 3种操作（查看/编辑/删除）
- 删除角色

#### 个人中心
- 查看和修改个人基本信息
- 修改登录密码
- 查看系统技术信息
