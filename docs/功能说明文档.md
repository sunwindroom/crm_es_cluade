# CRM v2.0 功能说明文档

## 角色权限体系

### 角色层级（由高到低）

| 角色 | 层级 | 核心权限 |
|------|------|---------|
| 系统管理员 | 最高 | 全部权限，唯一可删除数据 |
| 总裁 | 90 | 查看/编辑所有，转化线索，审批合同 |
| 营销副总裁 | 80 | 管理销售全流程，转化线索 |
| 技术副总裁 | 80 | 管理项目，指派项目经理，查看所有 |
| 财务 | 70 | 确认回款，查看全部财务数据 |
| 商务 | 70 | 管理合同，查看所有 |
| 销售经理 | 60 | 管理自己和下级销售的数据 |
| 销售 | 40 | 只能看自己负责的数据 |

### 数据可见性规则

- **销售**：只能看自己负责的线索/客户/商机
- **销售经理**：可看自己和所有下级销售的数据
- **技术副总裁/财务/商务/营销副总裁/总裁/系统管理员**：可查看全部数据

### 删除保护

- 线索、客户、商机一旦录入，**仅系统管理员**可删除
- 总裁/营销副总裁/系统管理员可编辑任意记录
- 其他角色只能编辑自己负责的数据

## 功能模块

### 1. 线索管理

- **新建**：所有人均可新建线索
- **分配**：上级可将线索分配给下级（仅能分配给自己的下级）
- **跟踪记录**：上下级均可添加跟进记录，均可查看全部跟进记录
- **转化**：总裁/营销副总裁可将线索转化为客户或商机
- **删除**：仅系统管理员可删除

### 2. 客户管理

- **新建**：所有人均可新建客户
- **分配**：上级可将客户分配给下级
- **拜访记录**：上下级均可填写，均可查看
- **删除**：仅系统管理员可删除
- **离职移交**：可将离职人员客户移交给接收人

### 3. 商机管理

- **新建**：所有人均可新建商机
- **分配**：上级可将商机分配给下级
- **跟进记录**：上下级均可填写，均可查看
- **赢单转项目**：赢单阶段商机可一键转化为项目（需总裁/营销VP/技术VP权限）
- **删除**：仅系统管理员可删除

### 4. 项目管理

- **新建**：总裁/技术VP/营销VP/商务/系统管理员可新建
- **指派PM**：技术副总裁负责指派项目经理
- **指派销售**：营销副总裁负责指派销售参与协调
- **任务管理**：项目经理选择参与人员，分配任务
- **工时填报**：项目参与人员每周填报工时，自动统计
- **里程碑**：项目经理更新里程碑状态
- **甘特图**：自动根据任务日期生成甘特图
- **进度自动计算**：任务进度更新后自动计算项目整体进度

### 5. 合同管理

- **新建**：总裁/技术VP/营销VP/商务/系统管理员可新建
- **回款节点**：新建合同必须至少设置1个回款时间节点
- **多期回款**：支持添加多个回款节点（首款/中期/尾款等）
- **差额提示**：实时显示节点金额合计与合同金额的差额

### 6. 回款管理

- **回款计划预警**：Tab页显示所有待付款节点，颜色区分紧迫程度
  - 🔴 红色：已逾期
  - 🟠 橙色：7天内到期
  - 🟡 黄色：30天内到期
  - 🔵 蓝色：正常
- **关联回款节点**：登记回款时可关联对应的合同回款节点
- **财务确认**：仅财务/系统管理员可确认回款

### 7. 离职移交

- 标记员工离职（禁用账号）
- 选择接收人和移交类型（线索/客户/商机/项目/合同）
- 接收人可查看离职人员的所有历史记录
- 完整移交日志记录

### 8. 报表统计

- **销售概览**：商机阶段分布、销售漏斗、线索来源转化率
- **客户分析**：来源分布、行业分布、月度新增趋势
- **回款分析**：12个月回款趋势、月度明细含环比
- **员工业绩**：赢单/回款金额对比、排行榜含角色信息
- **项目工时**：各项目总工时、人工成本、预算占比

### 9. 工作台

- 8大KPI卡片（含逾期回款节点预警）
- 回款预警提示（30天内到期节点）
- 12个月回款趋势图
- 客户行业分布饼图
- 销售漏斗可视化
- 员工业绩排行
