# CRM 客户关系管理系统 v2.0

完整企业级 CRM 系统，覆盖线索→客户→商机→项目→合同→回款全流程，含完整角色权限体系。

## 技术栈
- **前端**：React 18 + Vite + Tailwind CSS + Recharts + Zustand
- **后端**：Node.js + Express + better-sqlite3 + JWT
- **数据库**：SQLite（内嵌，无需单独安装）

## 快速启动

### 启动后端
```bash
cd backend
npm install
cp .env.example .env
npm start
```

### 启动前端
```bash
cd frontend
npm install
npm run dev
```

访问 http://localhost:5173

## 主要新增功能（v2.0）

### 1. 完整角色权限体系
- 8种预设角色：系统管理员、总裁、营销副总裁、技术副总裁、销售经理、销售、财务、商务
- 层级数据可见性：销售只看自己的，销售经理看自己和下级的，高管看全部
- 删除保护：线索/客户/商机一旦录入，仅系统管理员可删除
- 编辑保护：只有总裁/营销副总裁/系统管理员可编辑任意记录

### 2. 线索管理增强
- 线索跟踪记录：所有人可以添加，所有相关人员可查看
- 分配功能：上级可将线索分配给下级
- 转化控制：只有总裁/营销副总裁可转化线索为客户/商机

### 3. 项目管理增强
- 甘特图：自动根据任务日期生成可视化甘特图
- 里程碑管理：项目经理可添加/更新里程碑状态
- 工时记录：项目成员每周填报工时，自动统计工时和成本
- 进度自动计算：根据任务进度自动更新项目整体进度
- 指派项目经理：技术副总裁专属权限
- 商机转项目：赢单商机可一键转化为项目

### 4. 合同管理增强
- 必须设置回款节点：新建合同时必须至少添加1个回款时间节点
- 回款节点差额提示：实时显示节点金额合计与合同金额的差额

### 5. 回款管理增强
- 回款计划预警Tab：显示所有待付款节点的倒计时
- 颜色区分：红色（逾期）、橙色（7天内）、黄色（30天内）、蓝色（正常）
- 关联回款节点：登记回款时可关联对应的合同回款节点

### 6. 离职移交（新功能）
- 标记用户离职：将账号状态设为离职
- 数据移交：将离职人员的线索/客户/商机/项目/合同批量转移给接收人
- 移交记录：完整记录每次移交操作

### 7. 报表增强
- 新增项目工时统计报表
- 工作台新增回款预警提示
- 新增30天内到期节点统计
